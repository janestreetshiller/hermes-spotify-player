# Security policy

## Supported version

Only the latest tagged release is supported.

## Reporting a vulnerability

Use this repository's **Security → Report a vulnerability** flow so details stay private. Do not open a public issue for a suspected credential leak, command-injection path, OAuth weakness, or cross-plugin access bug.

Include the plugin version, Hermes version, macOS version, reproduction steps, and the smallest safe proof of concept. Never include live Spotify tokens or client credentials.
